StockBuddy was built with PHP and MySQL. Login, registration, the portfolio dashboard, and admin features all run server-side on a PHP runtime with a local database.

This deployment is hosted on Netlify, which serves static files only and does not execute PHP or run MySQL. Only the landing page works here.

To use the full app, run it locally with XAMPP/MAMP, or deploy it to a PHP-capable host.
